1、重装系统

1.1、备份环境变量的注册表

​ 备份环境变量的目的是让你操作系统之后不需要再去配置环境变量

​ Win+R输入regedit

​ 打开下面这个地址

​ 系统环境变量位置

​ HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment

​ 用户环境变量位置

​ HKEY_CURRENT_USER\Environment

​ 分别用户环境变量和系统环境变量在Environment上右击点击导出就好了

​ 记得备份好，不要重装完系统备份的东西没了，重装系统之后双击就行了

​ 建议将软件都装在D盘，如果你只有一个盘当我没说

7、配置wls2 与windows主机的连通性

​ powershell 管理员运行

New-NetFirewallRule -DisplayName "WSL" -Direction Inbound  -InterfaceAlias "vEthernet (WSL)"  -Action Allow

​ 如果还不能ping通说明windows 主机拒绝了icmp报文

配置防火墙规则

​ 入站规则—>核心网络诊断-ICMP回显请求(ICMPv4-In)勾选启用就行了

clash for windows 设置

​ 打开 Allow LAN，

​ 在wsl中环境变量，统计下面代码，添加到.bashrc这个文件最后，

port=25648
hostip=$(cat /etc/resolv.conf | grep nameserver | awk '{ print $2 }')
# setp是设置代理的命令，可以自定义
alias setp='export https_proxy="http://${hostip}:${port}";export http_proxy="http://${hostip}:${port}";export all_proxy="socks5://${hostip}:${port}";export ALL_PROXY="socks5://${hostip}:${port}";echo "set proxy:";env|grep -i proxy'
# unsetp是取消代理的命令，可以自定义
alias unsetp='unset https_proxy; unset http_proxy; unset all_proxy; unset ALL_PROXY;echo "clear proxy:";env|grep -i proxy'
